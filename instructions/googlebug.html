Google Drive bug

Note: This bug was supposedly fixed.

This was an intermittent bug with Google Drive - if you waited a day it might go away again.

The only other way I know around this is to not use Google Drive - download the .zip and run the map generator locally from your hard drive instead, or run it from another service like dropbox.

Advanced details

The initial GET request to Google Drive is met with a "301 Moved Permanently" redirect to a new URL, but the redirect drops all the URL parameters from the request.

There is some discussions of the problem here:

  • https://code.google.com/p/google-apps-script-issues/issues/detail?id=4084
  • http://stackoverflow.com/questions/24188499/google-drive-broken-links-for-ajax-for-chrome-and-safari
