Exporting from a seed

How to automatically get all the locations

I've modified a version of AMIDST so that it can export location files for any world you have the seed to. You can download the modified version from this page.

It can also export ocean-maps, which means you'll be able to have the ocean shown on the parchment of the map once I release the next version (v1.7)

Requirements

  • 800 MB of RAM - This is because it generates a large area of map before exporting the locations.
  • A version of Minecraft installed on the computer. The AMIDST Exporter will list all the versions it can find on the computer, pick the one the map was originally generated with, or something close.

Using Amidst Exporter

  • When prompted, click the ⧁ button for the version of Minecraft the map was originally generated with.
  • Select: File → New → From seed, then follow the prompts to enter the seed for your world and the type of world.
  • Once the world has loaded, select: File → Export → Locations.

OS X users

The .jar file should run on a Mac, I apologise for not providing a native OS X app to download - I don't currently have access to a Mac.

To launch in OS X, open a terminal window and type:

java -Xmx1024M -jar AmidstExporter.jar
